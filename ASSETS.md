# ASSET BREAKDOWN — Lagokefalos

Base canvas: **1280×720**. Όλα PNG με transparency, σε **2x** μέγεθος από τα νούμερα του καμβά για crisp εμφάνιση (τα βάζουμε με scale 0.5). Τα keys ΔΕΝ αλλάζουν — αντικατάσταση στο `BootScene.js` (οδηγίες στο README).

## Backdrop (προαιρετικά — τώρα γίνονται procedural)

| Τι | Σημειώσεις |
|---|---|
| Νησί/στεριά βάθους | Φαρδιά λωρίδα πάνω από τη γραμμή νερού, ~1280×110. Σαντορίνη vibes: λευκά σπιτάκια, μπλε τρούλος |
| Ουρανός | Απλό gradient — μπορεί να μείνει κώδικας |

| Βυθός/άμμος | Λωρίδα 1280×60 με υφή άμμου (προαιρετικό) |

## Ήχοι (αντικαθιστούν τα synthesized στο sfx.js)

| Function | Τι ακούγεται | Προτεινόμενο |
|---|---|---|
| `catch` | Πιάσιμο τενεκεδακιού | Pop/gulp, <0.3s |
| `combo` | Combo x2/x3 | Ανοδικό arpeggio/κουδουνάκι |
| `jump` | Jump/float | Whoosh με φυσαλίδες |
| `hit` | Χτύπημα (αγκίστρι/χελώνα) | Ouch/buzz, punchy |
| `splash` | Είσοδος/έξοδος από νερό, βυθός | Μικρό splash |
| `turtleWarn` | Προειδοποίηση "!" | Διπλό beep/alert |
| `gameOver` | Τέλος | Κατηφορικό jingle, 1.5s |
| `start` | Έναρξη | Χαρούμενο jingle, <1s |
| — (extra) | Ambient loop | Υποθαλάσσιο ambience, loopable — το κουμπώνω όταν το έχεις |

Format: MP3 ή OGG (και τα δύο ιδανικά για συμβατότητα), 44.1kHz.
